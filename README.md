# To Do List App
Приложение для управления задачами с возможностью создания вложенных подзадач.
## Установка

### `1. Клонируйте репозиторий:`
   ```bash
   git clone https://github.com/kexxxit/todo-list-app.git
   ```
```bash
cd task-manager
```
### `2. Установите зависимости:`
```bash
npm install
```
### `3. Запустите проект:`
```bash
npm start
```
